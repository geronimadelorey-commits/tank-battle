# 坦克大战 - Tank Battle

一个基于 HTML5 Canvas 的经典坦克大战游戏，使用纯 JavaScript 实现，无需任何外部依赖。

## 在线游玩

直接在浏览器中打开 `index.html` 即可开始游戏。

## 游戏截图

游戏包含经典的坦克大战玩法，有玩家坦克、敌人坦克、可破坏的砖墙、不可破坏的钢墙，以及需要保护的基地。

## 操作说明

| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `空格` | 射击 |
| `P` | 暂停/继续 |
| `R` | 重新开始 |

## 游戏特性

- **玩家坦克**：绿色坦克，3条生命，拥有无敌保护时间
- **敌人坦克**：自动AI巡逻和追踪玩家，分为普通、快速、强力三种类型
- **可破坏砖墙**：子弹可以击碎
- **不可破坏钢墙**：子弹无法穿透
- **基地**：金色菱形，必须保护不被敌人摧毁
- **粒子特效**：坦克爆炸、击中墙体的粒子效果
- **关卡系统**：消灭所有敌人自动进入下一关，难度递增
- **得分系统**：不同类型敌人分值不同

## 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- CSS3

## 本地运行

```bash
git clone https://github.com/你的用户名/tank-battle.git
cd tank-battle
# 直接在浏览器中打开 index.html
```

或者使用本地服务器：

```bash
npx serve .
```
